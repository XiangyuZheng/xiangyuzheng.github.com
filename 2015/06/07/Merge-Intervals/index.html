[Leetcode] Merge Intervals

##Question
Given a collection of intervals, merge all overlapping intervals.

For example,
Given [1,3],[2,6],[8,10],[15,18],
return [1,6],[8,10],[15,18].

##Solution
We need to firstly sort the intervals by start time. We can use Collections.sort() to do this and write a comstom Comparator.
Then we can merge the intervals.
