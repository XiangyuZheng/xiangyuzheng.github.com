  1. 1. Question
  2. 2. Solution
  3. 3. Java Code

Question

Given a string, determine if it is a palindrome, considering only alphanumeric characters and ignoring cases.

For example,
“A man, a plan, a canal: Panama” is a palindrome.
“race a car” is not a palindrome.

Solution

For this question, we should consider some special cases like empty strings and non-alphanumeric strings. Also, remember to convert the letters to lower cases before comparing. A possible solution is to use two pointers moving from the first char and the last char until they meet.
