[Leetcode] Longest Palindromic Substring

  1. 1. Question
  2. 2. Solution
  3. 3. #1. DP
  4. 4. #2. Expend from center

Question

Given a string S, find the longest palindromic substring in S. You may assume that the maximum length of S is 1000, and there exists one unique longest palindromic substring.

Solution

#1. DP

We can use DP.
P[i, j] = true if substring [i, j] is a palindrome.
P[i, i + 1] = char(i) == char(i+1)
P[i, j] = P[i + 1, j - 1] && char(i) == char(j)

The time and space complexity are both O(n^2).

#2. Expend from center

Suppose we can expend from a center to get a palindrome so that there are 2n-1 centers (some centers are between two chars). We try to expend the 2n-1 centers and find the longest one. In this way, the space complexity can be reduced to O(1).
