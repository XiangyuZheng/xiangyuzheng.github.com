Question

There are two sorted arrays A and B of size m and n respectively. Find the median of the two sorted arrays. The overall run time complexity should be O(log (m+n)).

Solution

This is a hard question. If m == n, then the question is easy. When m != n, we couldnâ€™t apply general binary search algorithm. Instead, we can use a findK algorithm.

To implement findK(), we get the k/2-th element in each array, if they are equal, we are done. If one is smaller than the other, we can get rid of the all the elements before the smaller one, so that we reduce k/2 elements. Now we can continue to look for the (k - k/2)-th element in the rest elements.
