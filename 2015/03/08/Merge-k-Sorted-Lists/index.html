Question

Merge k sorted linked lists and return it as one sorted list. Analyze and describe its complexity.

Solution

There are many ways to solve this question. Letâ€™s try to use a minimum heap to do this. The size of heap is k so that each insert or get minimum operation takes logk time. Hence, the time complexity is O(nklogk) and space complexity is O(k). Another point is that we can use PriorityQueue in Java to implement a minimum heap.
