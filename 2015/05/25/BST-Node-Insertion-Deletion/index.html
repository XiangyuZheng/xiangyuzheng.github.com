BST insertion and deletion

Java code

#Insertion

Insertion is easy, we just need to find the place according to the current position. Insertion may not maintain the balance of BST.

#Deletion

Deletion is complicated.

  1. If the node(node1) to be deleted has a left sub tree.
    1) find the max node(node2) in its left sub tree, delete it.
    2) Replace node1 with node2.
  2. If node1 has no left sub tree.
    1) Replace node1 with its right child.
