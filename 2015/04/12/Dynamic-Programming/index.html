Dynamic Programming

##When to use DP?

  1. Input can’t be sorted
  2. Find minimum/maximum result
  3. Check the feasibility
  4. Count all possible solutions (Note that finding all possible solutions is DFS, not DP).

##Some tips for using DP:

  1. For substring and string matching problems, it’s usually DP problems.
  2. When writing DP for string, we had better to use dp[len + 1].
  3. Think about the current result based on previous results.
