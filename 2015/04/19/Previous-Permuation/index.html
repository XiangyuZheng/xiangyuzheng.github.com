Question

Given a list of integers, which denote a permutation.

Find the previous permutation in ascending order.

Example
For [1,3,2,3], the previous permutation is [1,2,3,3]

For [1,2,3,4], the previous permutation is [4,3,2,1]

Solution

The same as “Next Permuation”
Steps:

  1. From right to left, find the first number that violate the decreasing order. Store it as partition number.
  2. From right to left, find the first number that is smaller than the partition number. Store it as change number.
  3. Swap partition number and change number.
  4. Reverse all the items after the original partition index.
